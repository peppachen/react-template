

| 特性             | **Redux**                           | **React Context**            |
| ---------------- | ----------------------------------- | ---------------------------- |
| **主要用途**     | 全局状态管理，适合复杂应用          | 轻量级状态共享，适合简单场景 |
| **状态更新方式** | 通过 action 和 reducer 触发状态更新 | 通过 setState 或其他函数更新 |
| **性能优化**     | Redux 默认使用浅比较，性能更优      | 可能因为频繁更新导致重新渲染 |
| 使用场景         | 适合复杂状态管理                    | 适合小规模状态共享           |
